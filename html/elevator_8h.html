int elevator_amIAtAnyFloor ( )

Checks whether the elevator is at a floor or inbetween floors. Note that this function does not check which floor the elevator might be at, only whether it's at one or not.

Returns
0 if between floors, 1 if at a floor. Note that it could be at any of the available floors if 1 is returned!
